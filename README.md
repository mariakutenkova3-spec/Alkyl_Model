Жидкофазное алкилирование бензола этан-этиленовой фракцией по технологии EBMax

Актуальность:
В настоящее время активно разрабатываются методы оптимизации промышленных процессов получения этилбензола. Однако многие производства работают с усредненными данными, не используя весь
потенциал оборудования. Эту проблему можно решить путем калибровки параметров под реальные условия процесса с помощью программного модуля. Одним из ключевых параметров, существенно влияющих на качество
конечного продукта, является температура. В рамках проекта не только разработана кинетико-термодинамическая модель процесса, но и опредена оптимальная температура в рабочем диапазоне процесса.

Расчетная часть:
Кинетическая модель включает в себя 10 реакций (5 прямых и 5 обратных):
[1] этилен + бензол = этилбензол - целевая реакция процесса
[2] пропилен + бензол = кумол - побочная реакция процесса
[3] этилен + толуол = этилтолуол - побочная реакция процесса
[4] бензол + этилбензол = дифенилэтан + водород - побочная реакция
процесса
[5] этилбензол+ этилен = диэтилбензол - побочная реакция процесса

Для этилена в системе дифференциальных уравнений учтена адсорбция на катализаторе. Константы скоростей прямых реакций рассчитываются по уравнению
Эйринга. Реакции рассматриваются как двухстадийные, а значения энтропий и энтальпий переходного состояния берутся из литературных
данных.Процесс считается адиабатическим, т.е. все тепло, выделяющееся в ходе реакции, расходуется на нагрев выходного потока. Для оценки адекватности в расчете также сводится материальный и тепловой баланс.

Программная часть:
Программа состоит из трех библиотек и файла запуска. Библиотеки созданы для удобства восприятия кода: в Alkyl_Consts заданы все константы, в Alkyl_Calc с помощью numpy и scipy реализованы все
вычисления (решение жесткой системы диф. уравнений методом Radau и определение оптимальной температуры), в Alkyl_Model с помощью matplotlib и tkinter создан интерфейс программы.
Результатом проделанной работы является приложение (exe-файл), которое по введенным параметрам (мольные доли бензола, этилена, пропилена и метилбензола; температура; давление; расход входного потока; время)
определяет константы скоростей всех реакций, итоговые концентрации всех компонентов, температуру продуктовой смеси на выходе, а также, выводит 4 графика: зависимости концентраций веществ от времени
(отдельно для основной и побочных реакций); зависимость выхода целевого продукта от температуры с определением оптимального режима; зависимость конверсий бензола и этилена от времени.
